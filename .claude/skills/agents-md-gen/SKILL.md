# AGENTS.md Generator

This skill generates AGENTS.md files that document AI agents in a repository according to AAIF standards.

## Usage
Run this skill when you need to create or update an AGENTS.md file for a repository. The skill will analyze the repository structure and generate appropriate documentation for AI agents.

## Process
The skill automatically:
1. Analyzes the repository structure
2. Identifies agent-related files and configurations
3. Documents repository conventions and guidelines
4. Generates a properly formatted AGENTS.md file

The generation logic is implemented in the supporting script to maintain consistency and follow AAIF standards.
